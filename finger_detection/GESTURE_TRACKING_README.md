# 手勢追蹤與臉部馬賽克系統

## 功能說明

系統會自動追蹤用戶比出的不雅手勢次數，當累積達到 **5 次**時，自動啟用**臉部馬賽克**功能。

- **不雅手勢 1-4 次**：只對手部馬賽克
- **不雅手勢 ≥5 次**：手部和臉部都會馬賽克
- **按 q 離開**：自動重置計數器

## 如何運行

### 啟動程式

```bash
python main.py
```

### 操作說明

1. 程式啟動後會顯示攝影機畫面
2. 偵測到不雅手勢會自動計數並對手部馬賽克
3. 達到 5 次後會啟用臉部馬賽克
4. 按 `q` 鍵退出程式（計數器會自動重置）

### 畫面資訊

左上角會顯示：
- `Bad Gestures: X/5` - 當前次數
- `Status: Normal` (綠色) - 正常狀態
- `Status: FACE MOSAIC ON` (紅色) - 臉部馬賽克已啟用

## 調整設定

如需修改閾值，編輯 `config.py`：

```python
BAD_GESTURE_THRESHOLD = 5  # 改成其他數字來調整閾值
```
